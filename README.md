# Resumen de etiquetas HTML usadas en el formulario:

<html>, <head>, <meta>, <title>, <link>, <body>: estructura básica del documento.

<h1>: título del formulario.

<form>: contenedor principal del formulario.

<div>: agrupación de campos.

<label>: etiquetas descriptivas para los campos.

<input>: campos de entrada (text, radio, email, tel, password, file, checkbox).

<select> y <option>: lista desplegable de países.

<textarea>: campo de texto para mensajes.

<button>: botones de envío y reinicio.

<br>: saltos de línea para separación visual.
